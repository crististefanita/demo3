# source-delta-carrier

Acest pachet nu vinde orders empty sau invoices empty ca adevar principal.
Diff-ul compilat arata o familie autentificata de checkout construita ca sa mute ruptura mai sus:
payment success -> missing commercial identity -> no persistent bridge into orders / invoices after relogin.
Run diff-urile raman deltele minime per claim; acest carrier doar explica de ce root diff-ul trebuie citit ca familie coerenta si nu ca cinci replay-uri izolate.
